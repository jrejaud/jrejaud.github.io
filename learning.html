Fundamentals of programming with Python

Prerequisites: None

No idea how to write a computer program? I recommend starting with Python with Codecademy.

Python is, in my opinion, the easiest programming language to learn. It's also useful! Youtube and Dropbox were written in Python.

Codecademy also contains a ton of other beginner tutorials for other languages like Ruby, Php, JS, HTML, ect.

Fundamentals of Physics and Maths

Prerequisites: None

No background in Physics and Maths? KhanAcademy's Physics and Maths guides are very helpful.

Do you know algebra? Does coulomb's law sound familiar? Do you even know how to Magnetism, bro? If not, these tutorials should be edifying.

If you want to half-ass your way to the finish line learn the very basics you need to get by, you should understand (at a minimum) Electricity and Magnetism and up to Algebra before proceeding.

If you want to become Tony Stark and Nikola Tesla actually understand wtf is going on (also known as "Engineering") you should understand the fundamentals of physics and up to Multivariable Calculus.

Fundamentals of Electronics

Prerequisites: Fundamentals of Physics and Maths

Now things are getting interesting Soon you will be able to build the DeathStar!. At this point you should know how to magnetism. Electronics is applied electrical (and arguably mechanical) Physics.

Sparkfun has excellent tutorials, the categories of relevance (in order) are concepts (electronics theory), technology (what is a resistor, teach me how to capacitor, ect.) and skills (how to use a multimeter, ect.)

I highly recommend going through ALL of the concept, technology, and skills tutorials. Will that take a long time? Yes!

Arduino: Electronics Prototyping Platform

Prerequisites: Python (or any other programming language), Fundamentals of Electronics (Can be done in parallel with Arduino tutorials)

The Arduino board is an open-source electronics prototyping platform. It's a physical board where you can plug lights and motors and radio modules and all kinds of crazy crap!

Ready to start building electronics? I recommend buying a starter kit from Adafruit or any other place. Acquire an Arduino Uno Board to start and some random electronics components (LEDs, motors, ect).

Start with Adafruit's Lesson 1 and then complete the other Lessons.

At this stage, you now have enough knowledge to start building robots! Congratulations!

Raspberry Pi: Micro-PC

Prerequisites: Python (optional but recommended), familiarity with Arduino, familiarity with Linux (No tutorial here yet, go on google)

If an Arduino board lets you make robots and electronics, then what is a Raspberry Pi? The Raspberry Pi is essentially a small computer running Linux.

Do you want to make something that drives around, an LED watch, do more "electronic" things? You might want to use an Arduino. Arduino code is written in C++.

Want to set up a server, do image processing, do more "computer" things? You might want to use a Raspberry Pi. A Raspberry Pi a cheap, tiny computer that runs an Operating System called Linux. You use it the way you would use your Laptop or Desktop.

A Raspberry Pi isn't a replacement or "upgrade" for an Arduino, it's a different type of platform to accomplish different things. However, there is some overlap and having Arduino experience is very helpful if you want to do "electronic" things with a Raspberry Pi.

Start with Adafruit's Lesson 1 and then complete the other Lessons.

Embedded Systems with Atmel and AVR-C

Prerequisites: Arduino (and a whole lot of patience)

IMHO, The Arduino Development Board excels in three areas: learning how to use microcontrollers, rapid initial prototyping, and quick "one-time" projects.

If you want to accomplish anything above and beyond those three things, I recommend you ascend to the more esoteric "microcontroller". For these tutorials, the target is the ATMega328p microcontroller (the one inside the Arduino Uno Board).

The ATMega328p is an AVR microcontroller manufactued by Atmel and is typically programmed in the C language. Unfortunately, there are not as many tutorials to programming AVRs and using the AVR-C library as there are for Arduino.

First, read After Arduino Part 1 and then complete Ladyada's tutorial, Hack a day's tutorial, and finally He killed my wire's tutorial.

For AVR-C Library references, read After Arduino Part 2.

How to make a website: HTML, CSS, jQuery, and Hosting

Prerequisites: None, Python or other programming experience will help

Want to learn how to make a website?

First, learn HTML and CSS. TLDR: HTML is content on a webpage and CSS is the style. To know more about the power of CSS, visit CSS Zen Garden.

jQuery lets you build interactive elements and animations.

If you want to host your website (for free), you can do so with GitHub pages.

Android Development

Prerequisites: Python (or any other programming language)

Android apps are written in the Java language.

First, learn the fundamentals of programming and then learn the specifics of Java at Jenkov.com's tutorials.

Once you have a good grasp of Java, Google's official Android Tutorials will make a lot more sense.

But you still need moar! Udacity's Android tutorial holds your hand less than the official documentation, but guides you through making your own app!
