Learning

Tutorials and guides on learning programming, electronics, and robotics.

This page also contains external resources are free or pay websites that I have personally used to learn about programming, electronics, history, literature, ect!

Want to learn how to build the DeathStar programming, electronics, and robotics?


Are you this dog? Then start here. If you know what you are doing then go straight to the external resources.


Fundamentals of programming with Python

Prerequisites: None

No idea how to write a computer program? I recommend starting with Python with Codecademy.

Python is, in my opinion, the easiest programming language to learn. It's also useful! Youtube and Dropbox were written in Python.

Fundamentals of Physics and Maths

Prerequisites: None

No background in Physics and Maths? KhanAcademy's Physics and Maths guides are very helpful.

Do you know algebra? Does coulomb's law sound familiar? Do you even know how to Magnetism, bro? If not, these tutorials should be edifying.

You should understand (at a minimum) Algebra and Electricity and Magnetism before proceeding.

Fundamentals of Electronics

Prerequisites: Fundamentals of Physics and Maths

Now things are getting interesting Soon you will be able to build the DeathStar!. At this point you should know how to magnetism. Electronics is applied electrical (and arguably mechanical) Physics.

Sparkfun has excellent tutorials, but I find their cataloging scheme is not very organized (You see, they skipped the maths tutorial and thus they do not put things in order: 1, 2, 3, ect.)

I'm currently working to re-catalogue their tutorials in an order which makes a bit more sense.

Learn how to use an Arduino Board (Now you can start to make the DeathStar robots and electronics!)

Prerequisites: Python (or any other programming language), Fundamentals of Electronics

Ready to start building electronics? Start learning Arduino with Adafruit Learning.

The Arduino board is an open-source electronics prototyping platform. That means you buy a starter kit from Adafruit or any other place and then start building things using those parts!

Start with Adafruit's Lesson 1 and then complete the other Lessons.

At this stage, you have enough knowledge to start building robots! Congratulations!
